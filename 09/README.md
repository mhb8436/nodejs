# Chapter 09: 프로젝트 구조화

이 디렉토리는 Node.js 프로젝트를 체계적으로 구조화하는 방법을 다루는 예제 코드를 포함하고 있습니다.

## 학습 목표

- 모듈화와 코드 구조화
- 디자인 패턴 적용
- 의존성 주입
- 미들웨어 구조화
- 라우터 모듈화
- 설정 관리
- 에러 처리 구조화
- 로깅 시스템 구현

## 디렉토리 구조

- **ch09_01.mongodb**: MongoDB를 사용한 데이터 모델링 예제
- **ch09_02/**: 기본 프로젝트 구조화 예제
- **ch09_03/**: 고급 프로젝트 구조화 패턴

## 사용 방법

1. 이 디렉토리로 이동합니다:

   ```bash
   cd 09
   ```

2. 필요한 의존성을 설치합니다:

   ```bash
   npm install
   ```

3. 예제를 실행합니다:
   ```bash
   # ch09_02 디렉토리로 이동
   cd ch09_02
   npm start
   ```

## 사전 준비

- Node.js가 설치되어 있어야 합니다
- npm이 설치되어 있어야 합니다
- Chapter 01-08의 내용을 이해하고 있어야 합니다
- MongoDB에 대한 기본 이해가 필요합니다
