# Chapter 4: Express.js 입문

이 챕터에서는 Express.js의 기본적인 사용법을 학습합니다.

## 예제 설명

1. `ch04_01.js`: Hello World 예제

   - Express.js의 가장 기본적인 사용법
   - 간단한 라우팅과 서버 시작 방법

2. `ch04_02.js`: 간단한 홈페이지 예제

   - 여러 페이지로 구성된 웹사이트 만들기
   - 정적 파일 제공
   - HTML과 CSS 통합

3. `ch04_03.js`: 데이터 표시 예제
   - JSON API 엔드포인트 구현
   - HTML과 API를 함께 사용하는 방법
   - 라우트 파라미터 사용
   - 에러 처리

## 실행 방법

1. 필요한 패키지 설치:

   ```bash
   npm install express
   ```

2. 각 예제 실행:

   ```bash
   node ch04_01.js
   node ch04_02.js
   node ch04_03.js
   ```

3. 브라우저에서 확인:
   - http://localhost:3000 접속

## 학습 포인트

- Express.js 서버 생성 및 실행
- 기본적인 라우팅 설정
- HTML 응답 전송
- JSON API 구현
- 정적 파일 제공
- 라우트 파라미터 사용
- 기본적인 에러 처리
